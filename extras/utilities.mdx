# Utility Commands

Everyday tools for members and moderators alike.

## Information

| Command | Description |
|---|---|
| `,userinfo [user]` | View detailed info about a user |
| `,serverinfo` | View server statistics |
| `,channelinfo [channel]` | View channel details |
| `,avatar [user]` | Get a user's avatar |
| `,banner [user]` | Get a user's banner |
| `,inviteinfo <invite>` | View info about an invite link |

## History & Tracking

| Command | Description |
|---|---|
| `,avatarhistory [user]` | View someone's past avatars |
| `,names [user]` | View someone's name history |
| `,seen <user>` | When was a user last active |
| `,firstmessage [channel]` | Jump to the first message in a channel |
| `,snipe` | View the last deleted message (alias: `,s`) |
| `,editsnipe` | View the last edited message (alias: `,es`) |

## Tools

| Command | Description |
|---|---|
| `,remind <time> <message>` | Set a reminder |
| `,poll <question>` | Create a poll |
| `,screenshot <url>` | Screenshot a webpage |
| `,color <hex>` | Preview a color |
| `,jumbo <emoji>` | Enlarge an emoji |
| `,steal <emoji>` | Steal an emoji from another server |
| `,mp3` | Convert an attachment to MP3 |
| `,shazam` | Identify a song from audio |
| `,transcribe` | Transcribe audio to text |

## External

| Command | Description |
|---|---|
| `,weather <location>` | Get the current weather |
| `,crypto <coin>` | Check cryptocurrency prices |
| `,valorant <user#tag>` | View Valorant stats |
| `,search <query>` | Search the web |

## Server Stats

| Command | Description |
|---|---|
| `,membercount` | Current member count |
| `,boosters` | List server boosters |
| `,bots` | List all bots in the server |
| `,roles` | List all server roles |
| `,emotes` | List all server emotes |
| `,invites` | View invite leaderboard |
| `,newmembers` | View recently joined members |
| `,oldest` | View the oldest members |
| `,topcommands` | Most used commands in the server |
