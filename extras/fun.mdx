# Fun Commands

Take a break from moderation and have some fun.

## Commands

| Command | Description |
|---|---|
| `,tictactoe @user` | Challenge someone to Tic-Tac-Toe |
| `,tictactoe statistics` | View your win/loss record |
| `,tictactoe leaderboard` | Server leaderboard |
| `,translate <text>` | Translate text to English (alias: `,tr`) |
| `,urbandictionary <word>` | Look up a word on Urban Dictionary (alias: `,urban`) |
| `,lyrics <song>` | Get lyrics for a song |
| `,quote` | Get a random quote |
| `,book <title>` | Search for a book |
| `,uwu <text>` | UwU-ify your text |
| `,choose <option1>, <option2>` | Let Teen choose for you |
| `,wikihow <query>` | Search WikiHow articles |
| `,ocr` | Extract text from an image (attach an image) |
