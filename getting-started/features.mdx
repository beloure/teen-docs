# Feature Overview

Teen packs **130+ commands** across moderation, music, server configuration, integrations, and utility — all in one bot.

## At a Glance

| Category | Highlights |
|---|---|
| **Security** | Antinuke, mass mention filter, word filter, automod |
| **Moderation** | Ban, kick, timeout, mute, purge, force nickname, hardban, mod stats |
| **Music** | Spotify/SoundCloud/YouTube playback, queue, filters, autoplay |
| **Server Setup** | Welcome/goodbye messages, autorole, starboard, voicemaster, tickets, logging |
| **Integrations** | Last.fm (nowplaying, stats, whoknows), TikTok auto-embed |
| **Fun** | Translate, urban dictionary, TicTacToe, lyrics, quotes, uwu |
| **Utilities** | Avatar history, reminders, polls, screenshots, weather, Valorant stats, Shazam |
