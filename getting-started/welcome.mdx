# Welcome to Teen

Teen is a powerful, all-in-one Discord bot built to handle everything from moderation and security to music, Last.fm, and server customization — so you don't need five different bots cluttering your server.

<Info>
  Your server prefix is set to `,` by default. Use `,prefix set (symbol)` to change it.
</Info>

## Quick Start

<Steps>
  <Step title="Invite Teen">
    [Add Teen to your server](https://discord.com/oauth2/authorize?client_id=YOUR_CLIENT_ID&permissions=8&scope=bot) with the required permissions.
  </Step>
  <Step title="Set your prefix">
    Run `,prefix set !` (or whatever symbol you prefer) to customize your prefix.
  </Step>
  <Step title="Run setup">
    Use `,setup` to quickly configure essential features like logging, welcome messages, and moderation.
  </Step>
  <Step title="Explore">
    Use `,help` to browse all available commands, or check the guides below.
  </Step>
</Steps>
