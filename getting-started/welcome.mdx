# What is Teen?

This guide will help you understand how to set up and use Teen. The only official invite link for the bot is [here](https://discord.com/oauth2/authorize?client_id=1477565161752625222&permissions=7417298934951415&scope=bot). If your questions aren't answered in these docs, join [discord.gg/sms](https://discord.gg/sms) and open a post.

<Warning>
  Make sure Teen's role is at the **very top** of your server's role list. This prevents permission issues and allows the bot to run moderation, antinuke, and management commands without errors.
</Warning>

## Setup

<Steps>
  <Step title="Position the bot role">
    Drag Teen's role to the **top** of your role hierarchy in Server Settings → Roles.
  </Step>
  <Step title="Grant all permissions">
    Ensure Teen has all the permissions it was invited with. Do not remove any of the default permissions from the bot role.
  </Step>
  <Step title="Run setup">
    Use `,setup` to configure core features — you'll be prompted to set up **VoiceMaster**, **Jail**, **Logging**, and **Moderation**.
  </Step>
  <Step title="Customize your server">
    Browse the docs below to fine-tune each feature to your liking.
  </Step>
</Steps>

<Info>
  Only the **server owner** can run the `,setup` command. As you go through this documentation, you'll learn which commands are restricted to server owners, antinuke admins, and other permission levels.
</Info>

## Prefix

The default prefix is `,` — you can change it at any time.

<AccordionGroup>
  <Accordion title=",prefix">
    View the current server prefix.
  </Accordion>

  <Accordion title=",prefix set <symbol>">
    Change the server-wide prefix. Requires **Administrator** permission.
  </Accordion>

  <Accordion title=",prefix self <symbol>">
    Set a personal prefix that only applies to you. Other members are unaffected.
  </Accordion>

  <Accordion title=",prefix remove">
    Remove your personal prefix and revert to the server default.
  </Accordion>
</AccordionGroup>
