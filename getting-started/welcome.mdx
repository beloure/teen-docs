# What is Teen?

This guide will help you understand how to set up and use Teen. If your questions aren't answered here, join [Teen ](https://discord.gg/sms)and open a post.

<Warning>
  Make sure Teen's role is at the **very top** of your server's role list. This prevents permission issues and allows the bot to run moderation, antinuke, and management commands without errors.
</Warning>

## Setup

<Steps>
  <Step title="Position the bot role">
    Drag Teen's role to the **top** of your role hierarchy in Server Settings → Roles.
  </Step>
  <Step title="Grant all permissions">
    Ensure Teen has **Administrator** permissions or all necessary permissions enabled.
  </Step>
  <Step title="Run setup">
    Use `,setup` to configure core features — you'll be prompted to set up **VoiceMaster**, **Jail**, **Logging**, and **Moderation**.
  </Step>
  <Step title="Customize your server">
    Browse the docs below to fine-tune each feature to your liking.
  </Step>
</Steps>

<Info>
  Only the **server owner** can run the `,setup` command. As you go through this documentation, you'll learn which commands are restricted to server owners, antinuke admins, and other permission levels.
</Info>

## Prefix

The default prefix is `,` — you can change it at any time.

| Command | Description |
| --- | --- |
| `,prefix set <symbol>` | Change the server-wide prefix |
| `,selfprefix <symbol>` | Set a personal prefix (only affects you) |
| `,selfprefix remove` | Remove your personal prefix |
