# Welcome to Teen

Teen is a multi-purpose Discord bot built for server management, music, integrations, and automation.

## Quick Start

1. [Join the support server](https://discord.gg/teen) and invite Teen to your server
2. Grant the bot **Administrator** permissions for full functionality
3. Use `,help` to view all available commands
4. Use `,help <command>` to get detailed info on any specific command

## Default Prefix

The default prefix is `,` (comma). You can change it per-server or set a personal prefix — see the [Prefix](/getting-started/prefix) page.

## Permissions

Most configuration commands require **Manage Server** or **Administrator** permissions. Moderation commands require their respective Discord permissions (e.g. Ban Members, Kick Members).

## Need Help?

Join the [support server](https://discord.gg/teen) for assistance.
