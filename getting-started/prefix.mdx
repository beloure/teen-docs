# Prefix

Teen uses `,` as the default prefix. You can change it server-wide or set a personal prefix that works only for you.

## Commands

| Command | Description | Permission |
|---|---|---|
| `,prefix` | View the current server prefix | — |
| `,prefix set <symbol>` | Change the server prefix | Administrator |
| `,selfprefix <symbol>` | Set a personal prefix (only applies to you) | — |
| `,selfprefix remove` | Remove your personal prefix | — |

## Examples

```
,prefix set !
,prefix set ?
,selfprefix ;
```

Your self-prefix overrides the server prefix for your commands only. Other members are unaffected.
