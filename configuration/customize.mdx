# Customize

Server owners can customize Teen's profile within their server, including the avatar, banner, and bio.

## Commands

| Command | Description | Permission |
|---|---|---|
| `,customize avatar [url]` | Set the bot's server avatar (attach image or provide URL) | Server Owner |
| `,customize banner [url]` | Set the bot's server banner | Server Owner |
| `,customize bio <text>` | Set the bot's server bio (max 190 characters) | Server Owner |
| `,customize reset` | Reset the bot's server profile to defaults | Server Owner |

## Examples

```
,customize avatar https://example.com/avatar.png
,customize bio The best bot in the server
,customize reset
```

You can also attach an image directly to the message instead of providing a URL.
