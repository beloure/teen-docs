# AutoMod

Teen's automod system includes a **word filter** and **mass mention filter**, both configured through the `,filter` command. Each filter can be set up with custom punishments, channel restrictions, and role exemptions.

## Opening the Filter Panel

```
,filter
```

This opens an interactive configuration panel where you can set up both the word filter and mass mention filter through a button-based UI.

## Word Filter

The word filter blocks messages containing specific words or phrases.

**Configuration options (via interactive panel):**

- **Toggle** — enable or disable the filter
- **Blocked Words** — add, remove, or view blocked words
- **Channels** — restrict to specific channels or apply to all
- **Exempt Roles/Users** — exclude certain roles or users from filtering
- **Punishment** — `ban`, `kick`, `strip`, `jail`, or `timeout`
- **Duration** — set timeout/jail duration when applicable

The word filter supports partial matching — blocking `badword` will also catch `mybadword123`.

## Mass Mention Filter

Prevents users from spamming mentions. Uses a **10-second rolling window** to detect mention spam across multiple messages.

### Commands

| Command | Description | Permission |
|---|---|---|
| `,filter massmentions` | Open the mass mention config panel | Administrator |
| `,filter massmentions on` | Enable the filter | Administrator |
| `,filter massmentions off` | Disable the filter | Administrator |
| `,filter massmentions threshold <number>` | Set the mention limit (1–50) | Administrator |
| `,filter massmentions channel <all \| #channel>` | Set filtered channels | Administrator |

Aliases: `massmention`, `mm`

### Configuration (via interactive panel)

Same options as the word filter: toggle, threshold, channels, exempt roles/users, punishment, and duration.

## Punishments

Both filters support the following punishments:

| Punishment | Description |
|---|---|
| `ban` | Permanently ban the user |
| `kick` | Kick the user from the server |
| `strip` | Remove all roles from the user |
| `jail` | Jail the user (requires jail setup) |
| `timeout` | Timeout the user for a set duration |
