# Audio Filters

Apply real-time audio effects to the currently playing track.

## Quick Toggles

These commands toggle their filter on or off:

| Command | Alias | Effect |
|---|---|---|
| `,nightcore` | — | Speed 1.25x, pitch 1.3x |
| `,bassboost` | `bb` | Heavy bass enhancement via equalizer |
| `,8d` | — | Rotating spatial audio (0.2 Hz) |

## Tune Command

Use `,tune` for full filter control:

| Command | Description |
|---|---|
| `,tune` | View active filter and list all presets |
| `,tune set <preset>` | Apply a filter preset |
| `,tune remove <preset>` | Remove a specific filter |
| `,tune clear` | Remove all active filters |

Aliases for clear: `reset`

Aliases for remove: `off`

## Available Presets

| Preset | Effect |
|---|---|
| `nightcore` | Speed 1.25x, pitch 1.3x |
| `slowed` | Speed 0.8x, pitch 0.85x |
| `vaporwave` | Speed 0.85x, pitch 0.75x |
| `8d` | Rotating audio (0.2 Hz) |
| `tremolo` | Wavering volume |
| `vibrato` | Wavering pitch |
| `karaoke` | Vocal reduction |
| `lowpass` | Muffle high frequencies |
| `bassboost` | Enhanced bass via equalizer |

## Examples

```
,tune set nightcore
,tune set slowed
,tune remove slowed
,tune clear
```

Only one filter can be active at a time. Use `,tune clear` to reset before switching presets.
