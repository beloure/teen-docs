# Queue Management

View and control the upcoming tracks in your server's queue.

## Commands

| Command | Description |
|---|---|
| `,queue` | View the current queue (up to 10 tracks shown) |
| `,queue remove <position>` | Remove a track by its position |
| `,queue move <from> <to>` | Move a track to a different position |
| `,queue shuffle` | Randomize the queue order |
| `,queue empty` | Clear the entire queue |

## Examples

**View the queue:**
```
,queue
```

**Remove the 3rd track:**
```
,queue remove 3
```

**Move track 5 to position 1 (play next):**
```
,queue move 5 1
```

**Shuffle everything:**
```
,queue shuffle
```
