# Permissions

Teen respects Discord's role hierarchy and requires specific permissions to function properly.

## Required Bot Permissions

For full functionality, Teen needs **Administrator** permission. At minimum, it needs:

- Manage Messages
- Manage Roles
- Manage Channels
- Ban Members
- Kick Members
- Moderate Members (for timeouts)
- Send Messages
- Embed Links
- Attach Files
- Connect & Speak (for music)

## Role Hierarchy

Teen can only moderate users whose **highest role** is below Teen's highest role. This means:

- Teen cannot ban/kick server owners
- Teen cannot assign roles that are above its own
- Antinuke requires Teen's role to be at the **very top** of the role list

<Warning>
  If Teen's role isn't high enough, moderation commands will silently fail or return a permission error. Always place Teen's role above any roles it needs to manage.
</Warning>

## Command Permissions

By default, moderation commands require the corresponding Discord permission:

| Command | Required Permission |
|---|---|
| `,ban` | Ban Members |
| `,kick` | Kick Members |
| `,timeout` | Moderate Members |
| `,purge` | Manage Messages |
| `,lock` | Manage Channels |
| `,role` | Manage Roles |
