# TikTok Integration

Teen can automatically fix and embed TikTok links so videos play directly in Discord.

## Commands

| Command | Description |
|---|---|
| `,tiktok setup` | Enable TikTok auto-embedding |
| `,tiktok disable` | Disable TikTok auto-embedding |

## How It Works

When a member posts a TikTok link, Teen automatically converts it into an embeddable format so the video plays inline — no need to click out of Discord.
