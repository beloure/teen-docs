# TikTok

Teen can automatically detect TikTok links in messages, download the video, and repost it with stats.

## Commands

| Command | Description | Permission |
|---|---|---|
| `,tiktok <username>` | Look up a TikTok profile | — |
| `,tiktok toggle` | Toggle auto-download of TikTok links | Manage Server |
| `,fyp [search]` | Get a random TikTok video | — |

Alias: `tt`

## Auto-Download

When enabled with `,tiktok toggle`, the bot will:

1. Detect TikTok URLs in messages
2. Delete the original message
3. Repost the video with view/like/comment stats

This works in all channels by default.

## FYP

Use `,fyp` to get a random TikTok video, or `,fyp <search>` to search for a specific topic.
