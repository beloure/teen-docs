# Last.fm

Link your Last.fm account to share what you're listening to, view stats, and compare taste with others.

## Connecting

```
,lastfm login
```

This sends you a DM with an authorization link. Click it to link your Last.fm account through OAuth.

To unlink:

```
,lastfm logout
```

## Now Playing

| Command | Alias | Description |
|---|---|---|
| `,nowplaying [member]` | `np`, `fm` | Show currently playing track |
| `,lastfm nowplaying [member]` | `lf np` | Same as above |

## Stats Commands

| Command | Alias | Description |
|---|---|---|
| `,lastfm toptracks [member]` | `lf tt` | Most listened tracks |
| `,lastfm topartists [member] [period]` | `lf artists` | Most listened artists |
| `,lastfm topalbums [member]` | `lf ta` | Most listened albums |
| `,lastfm count [member]` | — | Total scrobble count |
| `,lastfm streak [member]` | — | Current listening streak |
| `,lastfm playstrack <artist> - <track>` | `lf pt` | Play count for a specific track |
| `,lastfm playsalbum [member] <artist> - <album>` | `lf pla` | Play count for an album |
| `,lastfm playsall [member] <artist> - <album>` | `lf pa` | Plays for every track on an album |
| `,lastfm firstlisten <artist or artist - track>` | `lf first` | When you first played something |
| `,lastfm rediscover [member]` | — | Old songs you haven't played recently |
| `,lastfm recommendation [member]` | `lf rec` | Random artist recommendation |

### Time Periods

Some commands accept a time period:

| Period | Aliases |
|---|---|
| Overall | `alltime`, `all` |
| 7 days | `week`, `7day` |
| 1 month | `month`, `1month` |
| 3 months | `3month` |
| 6 months | `6month` |
| 12 months | `year`, `12month` |

## Server Commands

| Command | Description |
|---|---|
| `,lastfm whoknows <artist>` | Top listeners for an artist in this server |
| `,lastfm wkalbum <artist> - <album>` | Top listeners for an album in this server |
| `,lastfm crowns [member]` | View all artist crowns a user holds |
| `,lastfm mostcrowns` | Top crown holders in the server |
| `,lastfm taste <member> [period]` | Compare music taste with another member |

## Customization

| Command | Description |
|---|---|
| `,lastfm style <template>` | Set a custom now playing embed style |
| `,lastfm style reset` | Reset to the default embed |
| `,lastfm update` | Force update your account info |
