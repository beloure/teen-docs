# Autorole

Automatically assign roles when a member joins your server. Separate configurations for humans and bots.

## Commands

| Command | Description |
|---|---|
| `,autorole add <@role>` | Add a role to be assigned on join |
| `,autorole remove <@role>` | Remove an autorole |
| `,autorole humans <@role>` | Only assign to human members |
| `,autorole bots <@role>` | Only assign to bot accounts |
| `,autorole list` | View all configured autoroles |

## Examples

```
,autorole add @Member
,autorole humans @Verified
,autorole bots @Bot Role
```
