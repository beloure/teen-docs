# Tickets

Create a ticket system so members can open private support channels with your staff.

## Commands

| Command | Description |
|---|---|
| `,ticket setup` | Set up the ticket system with a panel |
| `,ticket add <@user>` | Add a user to the current ticket |
| `,ticket remove <@user>` | Remove a user from the current ticket |
| `,ticket close` | Close the current ticket |
| `,ticket category <category>` | Set the category for new tickets |
