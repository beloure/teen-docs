# Tickets

Create a ticket system so members can open private support channels with your staff.

## Setup

```
,ticket setup
```

This launches an interactive wizard that walks you through:

1. **Support Role** — which role can manage tickets
2. **Category** — where ticket channels are created
3. **Log Channel** — where ticket transcripts are sent
4. **Panel Message** — default text, custom text, or a custom embed
5. **Panel Type** — buttons or a dropdown menu
6. **Options** — configure each button/dropdown option with a label, emoji, description, and per-option ticket message

Requires **Manage Server** permission.

## Ticket Commands

These commands are used inside an open ticket channel:

| Command | Description |
|---|---|
| `,ticket close` | Close the current ticket |
| `,ticket add <user>` | Add a user to the ticket |
| `,ticket remove <user>` | Remove a user from the ticket |
| `,ticket rename <name>` | Rename the ticket channel |
| `,ticket claim` | Claim the ticket (assign yourself as handler) |
| `,ticket unclaim` | Unclaim the ticket |

Alias: `t`

Ticket transcripts are saved automatically when a ticket is closed and sent to the configured log channel.
