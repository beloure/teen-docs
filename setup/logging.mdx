# Logging

Teen can log server events to designated channels. Setup is done through an interactive wizard.

## Setup

```
,log setup
```

This launches a guided setup with two modes:

- **One Channel** — all events are logged to a single channel
- **Multiple Channels** — each event category gets its own channel (auto-created or manually assigned)

Requires **Manage Server** permission.

## Event Categories

| Category | Events |
|---|---|
| **Moderation** | Ban, Unban, Kick, Timeout, Untimeout |
| **Messages** | Delete, Bulk Delete, Edit, Pin, Unpin |
| **Members** | Join, Leave, Nickname Change, Role Add/Remove, Avatar Change, Username Change, Boost, Unboost |
| **Voice** | Join, Leave, Move, Server Mute/Deafen, Self Mute/Deafen, Stream Start/End, Camera On/Off |
| **Channels** | Create, Delete, Name Change, Topic Change, Slowmode Change, NSFW Toggle, Permission Changes |
| **Threads** | Create, Delete, Archive, Unarchive, Name Change, Member Join/Leave |
| **Roles** | Create, Delete, Name/Color/Permission/Hoist/Mentionable Changes |
| **Server** | Name, Icon, Description, Vanity, Verification Changes, Emoji/Sticker Add/Remove |
| **Invites** | Create, Delete |
| **Stages** | Open, Close, Topic Change |
| **Scheduled Events** | Create, Delete, Update, User Join/Leave |
| **Automod** | Rule Create/Delete/Update, Message Blocked, Member Timeout/Flagged |
| **Webhooks** | Create, Delete, Update |
| **Integrations** | Bot Add/Remove, Integration Add/Remove/Update |

Events are stored internally as `Module.Event` (e.g., `Messages.Delete`).
