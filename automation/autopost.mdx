# Auto Post

Schedule messages to be posted automatically at a set interval. Supports plain text, embeds, and tagscript variables.

## Commands

| Command | Description | Permission |
|---|---|---|
| `,autopost add <interval> <#channel> <content>` | Create a new autopost | Manage Server |
| `,autopost delete <id>` | Delete an autopost | Manage Server |
| `,autopost edit <id> [interval] [#channel] [content]` | Edit an existing autopost | Manage Server |
| `,autopost pause <id>` | Pause an autopost | Manage Server |
| `,autopost resume <id>` | Resume a paused autopost | Manage Server |
| `,autopost view <id>` | Manually trigger an autopost once | Manage Server |
| `,autopost list` | View all autoposts | Manage Server |

## Intervals

Use duration format: `30s`, `10m`, `2h`, `1d`. Minimum interval is **1 minute**.

## Examples

```
,autopost add 1h #general Remember to check out the rules!
,autopost add 6h #announcements {embed}$v2
$title: Daily Reminder
$description: Don't forget to bump the server!
$color: #16A34A
{/embed}
```

## Managing Autoposts

Each autopost is assigned an ID shown in `,autopost list`. Use the ID to edit, pause, resume, or delete it.

```
,autopost pause 3
,autopost resume 3
,autopost delete 3
```

Autoposts support [embed scripting](/extras/embed-scripting) for rich formatting.
